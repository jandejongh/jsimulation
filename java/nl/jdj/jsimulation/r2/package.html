A small package for discrete-event simulation.

It uses three important concepts:

  • Events (class SimEvent): an event represents a tuple of a time value and a corresponding action to be taken at that time;
  • Actions (interface SimEventAction): an action to be taken when an event is executed;
  • Event Lists (class SimEventList): a list of events ordered increasing in time.

The SimTimer class allows convenient scheduling of timer events on an event list.

Example code is available in the nl.jdj.jsimulation.r2.example package.
